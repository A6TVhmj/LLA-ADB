# LLA-ADB (Lemon Link Assistant 6.0)

Lemon Link Assistant 6.0 - ADB文件上传器

## 项目简介

LLA-ADB (Lemon Link Assistant 6.0) 是一个功能丰富的 Android 设备管理工具，主要包含以下功能：

1. ADB 文件上传器：通过 ADB (Android Debug Bridge) 将文件上传到 Android 设备
2. 网易云音乐下载器：可以从网易云音乐下载音乐文件
3. Bilibili 视频下载器：可以从 Bilibili 下载视频文件

## 功能特性

- ✅ **ADB 文件上传**：通过 ADB 协议安全高效地上传文件到 Android 设备
- 🎯 **图形化界面**：直观的用户界面，操作简单便捷
- 📁 **灵活路径选择**：支持自定义目标路径，满足不同上传需求
- 🎵 **网易云音乐集成**：内置音乐下载功能，一键获取喜爱歌曲
- 📺 **Bilibili 视频支持**：集成视频下载功能，轻松保存精彩内容
- 🔍 **设备自动检测**：智能识别已连接的 Android 设备
- 📊 **实时进度显示**：清晰展示上传进度和状态信息
- 🚀 **批量处理**：支持同时上传多个文件，提升工作效率

## 使用指南

1. **下载应用**：访问 [Releases](https://github.com/A6TVhmj/LLA-ADB/releases) 页面，下载最新版本的 `LLA-6.x.x.zip` 安装包
2. **解压运行**：解压下载的压缩包，双击 `LLA.exe` 启动应用程序
   > **注意**：首次启动时，程序会自动开启 ADB 服务，启动时间可能稍长（约10秒）
3. **连接设备**：使用 USB 数据线连接 Android 设备，并确保已开启 USB 调试模式
4. **开始使用**：在主界面选择所需功能（文件上传、音乐下载或视频下载），按提示操作即可
